Objectiu del joc

Aconseguir formar una jugada de valor superior a la dels contraris. No cal que la jugada sigui millor que la dels aquests, n'hi haurà prou que els rivals que creguin que ho és, perquè es retirin abans de veure les cartes.

Baralla de cartes

Es juga amb una baralla anglesa de 52 cartes. Se solen emprar dues baralles de diferent color de revers perquè, mentre 1 està en joc, l'altra quedi estudiada al costat de què ha de donar la propera vegada. Prèviament, els jugadors decidiran emprar un o els dos comodins, encara que el més recomanable és jugar sense cap comodí. El comodí és una carta que serveix per suplir o representar qualsevol altra carta.

Nombre de jugadors

Hi poden jugar de dos a set jugadors, tot i que les partides més interessants són les formades per cinc o sis. Cada jugador juga per a si, sense company.

Ordre i valor de les cartes

L'ordre, de major a menor, és el següent: es, rei (K), dama (Q), jota (J), deu, nou, vuit, set, sis, cinc, quatre, tres-dos. Els pals tenen tots igual valor en qualsevol de les jugades.

Requeriments funcionals

Express

• POST /iniciarJoc/codiPartida
• GET /obtenirCarta/codiPartida : JSON
• GET /mostrarCartes/codiPartida : JSON
• PUT /tirarCarta/codiPartida/carta
• PUT /moureJugador/codiPartida/aposta/quantitat
• PUT /moureJugador/codiPartida/passa
• DELETE /acabarJoc/codiPartida

JAX RS

• POST /iniciarJoc/codiPartida
• GET /obtenirCarta/codiPartida : JSON
• GET /mostrarCartes/codiPartida : JSON
• PUT /tirarCarta/codiPartida/carta
• PUT /moureJugador/codiPartida/aposta/quantitat
• PUT /moureJugador/codiPartida/passa
• DELETE /acabarJoc/codiPartida

SOAP

• iniciarJoc(codiPartida:enter):booleà
• obtenirCarta (codiPartida:enter):Carta
• mostrarCartes(codiPartida:enter):Carta[]
• tirarCarta(codiPartida:enter,carta:Carta)
• moureJugador(codiPartida:enter, aposta:booleà, quantitat:enter)
• acabarJoc(codiPartida:enter):booleà
